## 📖 프로젝트 소개
> 🌙 나만의 꿈을 기록하고 해몽하는 공간, DreamDiary 🌙

- 꿈속 이야기를 기록하고 해석하며 내면의 세계를 탐험할 수 있는 꿈 일기 앱입니다.
- 꿈을 다른 사람들과 공유하고 함께 해몽하며 다양한 해석을 나눌 수 있습니다.
